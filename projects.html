    Text Animation Editor

    I made a text animation program over the 23-24 school year. It has a multitude of animations and interfacing abilities. The entire thing was programmed by me based off with only basic rendering handled by p5.js. It's designed to be able to make animations for open slides or intergration into slide show for title transitions.

    Each of the animations are a hand crafted algorithm designed and implemenented by myself. You can see some of the products of this project through out the web site. The current public access version is avaliable Here
